# 作业提交系统

一个后端基于ASP.NET Core框架，前端基于VUE的简易课程设计，内容包含：  

- 首页：其他页面的入口  
- 作业提交页：表单提交和上传附件功能  
- 作业列表页：可分类查找和下载作业  

## 安装

### 后端：homework-server

安装依赖：

```js
dotnet add package
```  

起服务：

```js
dotnet watch run //带热更新
```

### 前端：homework-webapp

安装依赖：

```js
npm install
```

运行：

```js
npm run serve
```
